AndroidScreencast - download latest

This project gives the opportunity to use your phone even with a broken screen.

Features:

  • No client needed
  • Support for Tap and Swipe gestures
  • Write messages using PC keyboard
  • Support for landscape mode
  • Browse your phone files on PC

Prerequisites:

  • Installed Java with version >= 7
  • Any ADB for your OS (for example from Android SDK)

Running the app:

  • Point your adb path in app.properties file. Example app.properties content:
    `adb.path=/usr/bin/adb`
  • Start AndroidScreencast by running: `java -jar androidscreencast.jar`
  • Profit.
